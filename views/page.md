
```mermaid
stateDiagram-v2
    [*] --> 一覧表示
    
    一覧表示 --> 追加画面 : 追加ボタン
    一覧表示 --> 詳細表示 : 名称からリンク
    
    詳細表示 --> 編集画面 : 編集ボタン
    詳細表示 --> 削除画面 : 削除ボタン
    詳細表示 --> 一覧表示 : 一覧に戻る
    
    追加画面 --> 登録 : 登録ボタン (POST)
    編集画面 --> 登録 : 更新ボタン (POST)
    
    登録 --> 一覧表示 : 処理後にリダイレクト
    削除画面 --> 一覧表示 : 削除後に戻る
```
